emmit 실습

0. Tag name + Tab

1. child : >

ex) nav>ul>li


    <nav>
      <ul>
        <li>text</li>
        <li>text</li>
        <li>text</li>
      </ul>
    </nav>

Sibling : +

ex)div+p+bq 


      <div></div>
      <p></p>
      <blockquote></blockquote>

Climbup : ^

"Move up one level and write the following content."
ex)div+div>p>span+em^bq 


  <div>div</div>
  <div>
    <p><span></span><em></em></p>
    <blockquote></blockquote>
  </div>

Multiplication : *

ex)ul>li*5


  <ul>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
  </ul>

Grouping : ()

ex)div>(header>ul>li*2>a)+footer>p


  <div>
  <header>
    <ul>
      <li><a href=""></a></li>
      <li><a href=""></a></li>
    </ul>
  </header>
  <footer>
    <p></p>
  </footer>
</div>

Item numbering : $

'.' : class

ex) ul>li.item$*4


    <ul>
      <li class="item1"></li>
      <li class="item2"></li>
      <li class="item3"></li>
      <li class="item4"></li>
      <li class="item5"></li>
    </ul>

ex) ul>li.abc$*5


    <ul>
      <li class="abc1"></li>
      <li class="abc2"></li>
      <li class="abc3"></li>
      <li class="abc4"></li>
      <li class="abc5"></li>
    </ul>

tag_name [ attribute_name = attribute_value $ ] { content_text $ } * repeat_count 


  <p>
  <i>tag_name</i>                 <!-- 태그 이름 -->
  <strong>[</strong>              <!-- 속성 대괄호 -->
    <i>attribute_name</i>         <!-- 속성 이름 -->
    <strong>=</strong>            <!-- 등호 -->
    <i>attribute_value</i>        <!-- 속성 값 -->
    <strong>$</strong>            <!-- 반복 시 자동 번호 -->
  <strong>]</strong>              <!-- /속성 대괄호 -->
  <strong>{</strong>              <!-- 내용 중괄호 -->
    <i>content_text</i>           <!-- 내용 텍스트 -->
    <strong>$</strong>            <!-- 반복 시 자동 번호 -->
  <strong>}</strong>              <!-- /내용 중괄호 -->
  <strong>*</strong>              <!-- 반복 연산자-->
  <i>repeat_count</i>             <!-- 반복 횟수 -->
  </p>

[] : attribute, {} : content

ex)h[title=item$]{text $}*n


    <h title="abc" 1="">text 1</h>
    <h title="abc" 2="">text 2</h>
    <h title="abc" 3="">text 3</h>

ex)a[href=]{href $}*3

Since the **a** tag is an inline element, its results are displayed horizontally.


    <a href="">href 1</a><a href="">href 2</a><a href="">href 3</a>
  

    <a href="">href 1</a>
    <a href="">href 2</a>
    <a href="">href 3</a>
